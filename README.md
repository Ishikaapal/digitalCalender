📅 Digital Calendar – Interactive Wall Calendar UI

A modern, responsive digital wall calendar component built using React, Vite, and Tailwind CSS. This project recreates the aesthetic of a physical wall calendar while integrating interactive web features.

🚀 Live Demo

🔗 https://ishikaapal.github.io/digitalCalender/

🎯 Project Overview

This project is a frontend-focused implementation of an interactive calendar UI inspired by physical wall calendars. It emphasizes:

Clean and aesthetic design
Responsive layout
Component-based architecture
Smooth user experience

The design blends a hero image section with a structured calendar grid, creating a visually balanced and engaging interface.

✨ Features
📆 Calendar UI
Monthly calendar grid layout
Proper alignment of days and dates
Clean typography and spacing
🎨 Wall Calendar Design
Hero image section
Custom diagonal overlay using clip-path
Month and year display with visual hierarchy
⚛️ React-Based Architecture
Reusable components
State-driven UI rendering
Modular and scalable structure
📱 Fully Responsive
Desktop: side-by-side balanced layout
Mobile: stacked and optimized for touch
🛠 Tech Stack
⚛️ React (Vite)
🎨 Tailwind CSS
🧠 JavaScript (ES6+)
🌐 GitHub Pages (Deployment)
🧠 Key Concepts Used
React functional components & hooks
Component-based UI design
Tailwind utility-first styling
CSS clip-path for custom shapes
Responsive design (mobile-first approach)

⚙️ How to Run Locally
Clone the repository:
git clone https://github.com/ishikaapal/digitalCalender.git
Navigate into the project:
cd digitalCalender
Install dependencies:
npm install
Start development server:
npm run dev
